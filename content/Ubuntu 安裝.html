Ubuntu 安裝

先安裝 Virtualbox 和 Filezilla client

下載 Ubuntu 18.04.2 LTS 的安裝檔

下載 Ubuntu Server (18.04.2 LTS) 安裝檔

開啟 Virtualbox 

"New"  create virtual Machine

新建完成後

開啟"設定"

 

記住目前所啟用的網卡

(啟用中)

到剛新建好的 Virtual Machine 

settings ->Network

切換成 Bridged Adapter

指定到正在使用的網卡

 執行剛安裝完 Virtual Machine 

選擇下載下來的 Ubuntu Server (18.04.2 LTS) 安裝檔

開始安裝 Ubuntu Server

P.S. 在自己的電腦沒有 proxy 所以安裝 Ubuntu 時需要注意，proxy address 的部分就可以不用設定

安裝 Virtualbox (6.0.8) 載點 :

https://www.virtualbox.org/wiki/Downloads

下載 Ubuntu Server (18.04.2 LTS) 

https://www.ubuntu.com/download/server/thank-you?country=TW&version=18.04.2&architecture=amd64

安裝 Filezilla client 載點 :

https://filezilla-project.org/download.php?platform=win64
